Evaluation Results

—
Overall Score Average across all evaluated metrics
Faithfulness
Measures whether the generated answer is grounded in the retrieved context. A low score means the LLM is hallucinating — making claims not supported by your retrieved documents.
1.0 = fully grounded0.5 = some unsupported claims0.0 = hallucinating
Context Relevance
Measures how relevant the retrieved chunks are to the question. A low score means your retriever is fetching the wrong documents — tune your embedding model, chunking, or similarity threshold.
1.0 = highly relevant0.5 = partially relevant0.0 = irrelevant
Answer Relevance
Measures whether the answer actually addresses the question. An answer can be faithful to the context but still be off-topic or too tangential — this metric catches that.
1.0 = fully on-topic0.5 = partially relevant0.0 = irrelevant
Context Recall
Measures whether the retrieved context contains the information needed to produce the ground truth answer. A low score means the right information isn't in your knowledge base, or your retriever is failing to find it. Requires ground truth.
1.0 = context covers ground truth0.0 = missing key information
Context Precision
Measures the fraction of retrieved chunks that are actually useful. Retrieving 10 chunks but only 2 being relevant wastes tokens and confuses the LLM. Tune your top-k parameter if this is low.
1.0 = all chunks relevant0.5 = half relevant0.0 = none relevant
💡 How to interpret scores
80–100%Good — this dimension is working well.
50–79%Needs attention — investigate and improve.
0–49%Broken — likely causing poor user experience.

Install the library and add evaluation to your TypeScript project in minutes: 

npm install rageval @anthropic-ai/sdk
import Anthropic from '@anthropic-ai/sdk'
import { evaluate, faithfulness, answerRelevance } from 'rageval'

const results = await evaluate({
  provider: {
    type: 'anthropic',
    client: new Anthropic(),
    model: 'claude-opus-4-6',
  },
  dataset: [{
    question:    'What is the capital of France?',
    answer:      'Paris is the capital of France.',
    contexts:    ['France is a country in Europe. Its capital is Paris.'],
    groundTruth: 'Paris',
  }],
  metrics: [faithfulness, answerRelevance],
})

console.log(results.scores)
// { faithfulness: 0.97, answerRelevance: 0.95, overall: 0.96 }
